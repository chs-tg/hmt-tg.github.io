    What we're developing

    With the sponsorship of the Center for Hellenic Studies, we are maintaining an inventory of unique identifiers for for ancient Greek texts transmitted by manuscript copying. See the repository for the greekLit namespace for Canonical Text Services.

    While these identifiers are independent of any particular technology, their strictly defined syntax and semantics makes them ideal for machine processing with the CITE architecture. In contrast to the repository of identifiers for Greek texts, the CITE architecture is generically applicable, so we have made our work on it available through its own separate github organization: http://cite-architecture.github.io/.
